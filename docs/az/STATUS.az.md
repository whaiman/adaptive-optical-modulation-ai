# Layihənin Statusu

> 🌐 [English](../../STATUS.md) · [Русский](../ru/STATUS.ru.md)

**Cari mərhələ:** aktiv tədqiqat və inkişaf  
**Son yenilənmə:** 2026-04-04

İlkin prototipde konseptual problemlər aşkar edildikdən sonra layihə hazırda sıfırdan yenidən qurulur.

Məqsəd, Azad Məkanlı Optika (FSO) kanallarının fiziki davranışını düzgün əks etdirən neyron şəbəkəsi ilə idarə olunan Adaptiv Optik Modulyasiya kanalı yaratmaqdır.

---

## Tamamlananlar

- [x] Neyron şəbəkələrini adaptiv optik modulyasiya ilə birləşdirən ilkin prototip
- [x] Qarışıq reqressiya və klassifikasiya ilə ilkin təcrübələr
- [x] Arxitektura və konseptual problemlərin müəyyən edilməsi
- [x] Problemin fiziki cəhətdən əsaslı mərhələlərə yenidən formalaşdırılması

---

## İcra edilir

- [ ] Əvvəllər müəyyən edilmiş konseptual xətaların aradan qaldırılması üçün modelin yenidən işlənməsi
- [ ] FSO və adaptiv modulyasiya sahəsindəki müasir tədqiqat standartlarına uyğunlaşmaq üçün model imkanlarının genişləndirilməsi
- [ ] Reqressiya və klassifikasiyanı ayıran yeni arxitekturanın dizaynı
- [ ] Öyrətmə kanalının sıfırdan yenidən yazılması
- [ ] FSO kanal modelləşdirilməsi üçün fiziki cəhətdən düzgün giriş və çıxışların müəyyən edilməsi
- [ ] Reproduksiya edilə bilən təcrübə qurğusunun hazırlanması

---

## Planlaşdırılır

- [ ] Yekun AOM qərar kanalının tətbiqi (reqressiya → klassifikasiya → modulyasiya idarəetməsi)
- [ ] Süni intellektsiz adaptiv modulyasiya üsulları ilə müqayisəli qiymətləndirmə
- [ ] Tədqiqat məqaləsinin yenidən yazılması və nüfuzlu jurnal və ya konfransda dərc edilməsi
- [ ] Sənədləşdirmə və icra edilə bilən nümunələrin əlavə edilməsi
- [ ] Simulyasiya edilmiş və ya real FSO kanal məlumatlarının inteqrasiyası
