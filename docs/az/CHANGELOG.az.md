## [Buraxılmamış]

### Dəyişdirildi

- Neyron şəbəkəsi arxitekturasının tam yenidən dizaynı
- Tapşırıq ayrı reqressiya və klassifikasiya mərhələlərinə yenidən formalaşdırıldı
- Layihənin istiqaməti Azad Məkanlı Optikaya (FSO) doğru dəqiqləşdirildi
- İlkin prototipde müəyyən edilmiş konseptual xətaların davam edən düzəldilməsi
- FSO və adaptiv modulyasiya sahəsindəki müasir tədqiqat standartlarına uyğunlaşmaq üçün model imkanları genişləndirildi

### Silindi

- Konseptual məhdudiyyətlərə görə ilkin prototip arxitekturası
