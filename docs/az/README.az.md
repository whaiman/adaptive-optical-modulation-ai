# Süni İntellekt ilə Adaptiv Optik Modulyasiya

> 🌐 [English](../../README.md) · [Русский](../ru/README.ru.md)

> 📄 [CHANGELOG](CHANGELOG.az.md) · [STATUS](STATUS.az.md)

Bu repozitoriya **neyron şəbəkəsinin proqnozu ilə gücləndirilmiş adaptiv optik modulyasiyanın** istinad tətbiqini təqdim edir.
Layihə, optik modulyasiya parametrlərini dinamik olaraq proqnozlaşdırmaq və tənzimləmək üçün neyron şəbəkələrinin öyrədilməsini və tətbiqini nümayiş etdirir, ənənəvi üsullarla müqayisədə səmərəliliyi və dəqiqliyi artırmağı hədəfləyir.
Buradakı bütün kod orijinaldır və həm neyron şəbəkəsinin öyrədilməsi, həm də optik modulyasiya alqoritmi üçün tam reproduksiya oluna bilən iş axını təmin edir.  
Bu repozitoriya optik modulyasiya, siqnal emalı və süni intellektlə gücləndirilmiş idarəetmə sistemləri ilə maraqlanan tədqiqatçılar, mühəndislər və həvəskarlar üçün nəzərdə tutulub.

Metod, optik siqnalların real vaxt rejimində optimallaşdırılmasını təmin etmək üçün klassik adaptiv optik modulyasiyanı süni intellektə əsaslanan proqnozlaşdırma ilə birləşdirir.
Bu tətbiq, atmosfer turbulentliyi və düz görünüş xəttinin dəyişkənliyi kimi problemləri ön plana çıxararaq, fiber-optik sistemlər əvəzinə xüsusi olaraq **Azad Məkanlı Optika (AMO) tətbiqlərinə** yönəldilmişdir.
Adaptiv optik modulyasiya əvvəllər tədqiq edilmiş olsa da, bu repozitoriya **süni intellektə əsaslanan proqnozu inteqrasiya edən ilk açıq mənbəli istinad tətbiqini** təqdim edir.  
Bütün nəticələr, metriklər və nümunələr metodun başa düşülməsini, reproduksiyasını və genişləndirilməsini asanlaşdırmaq üçün təqdim edilmişdir.

Bu repozitoriyada həyata keçirilən konsepsiyalar ilk dəfə **Azərbaycan Texniki Universitetində** keçirilmiş **"İnfokommunikasiya Sistemləri və Süni İntellekt Texnologiyaları" mövzusunda 2-ci Beynəlxalq Elmi-Texniki Konfransda (4 dekabr 2024-cü il)** təqdim edilmişdir. O vaxt iş geniş diqqət çəkməsə də, bu repozitoriya həmin ideyaların tam açıq tətbiqini təmsil edir.  

Layihə **AGPL-3.0 lisenziyası** altında buraxılmışdır; bu, həmin kod əsasında yaradılan hər hansı törəmə iş, xidmət və ya tətbiqin açıq mənbəli qalmasını və orijinal mənbəyə düzgün istinad edilməsini təmin edir.  
Bu metodun töhfələri, eksperimentləri və genişləndirilmələri layihənin açıq mənbəli xarakterini qorumalıdır.
