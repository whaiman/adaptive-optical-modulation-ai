# Статус проекта

> 🌐 [English](../../STATUS.md) · [Azərbaycanca](../az/STATUS.az.md)

**Текущая стадия:** активные исследования и разработка  
**Последнее обновление:** 2026-04-04

После выявления концептуальных проблем в начальном прототипе проект в настоящее время перестраивается с нуля.

Цель — разработать конвейер адаптивной оптической модуляции на основе нейронных сетей, который корректно отражает физическое поведение каналов оптики свободного пространства (FSO).

---

## Выполнено

- [x] Начальный прототип, объединяющий нейронные сети с адаптивной оптической модуляцией
- [x] Ранние эксперименты со смешанной регрессией и классификацией
- [x] Выявление архитектурных и концептуальных проблем
- [x] Переформулировка задачи в физически обоснованные этапы

---

## В процессе

- [ ] Переработка модели для устранения ранее выявленных концептуальных ошибок
- [ ] Расширение возможностей модели для соответствия современным стандартам исследований в области FSO и адаптивной модуляции
- [ ] Проектирование новой архитектуры с разделением регрессии и классификации
- [ ] Переписывание конвейера обучения с нуля
- [ ] Определение физически корректных входных и выходных данных для моделирования FSO-канала
- [ ] Подготовка воспроизводимой экспериментальной установки

## Запланировано

- [ ] Реализация финального конвейера принятия решений AOM (регрессия → классификация → управление модуляцией)
- [ ] Сравнительная оценка с методами адаптивной модуляции без ИИ
- [ ] Переработка и публикация статьи в рецензируемом журнале или на конференции
- [ ] Добавление документации и запускаемых примеров
- [ ] Интеграция симулированных или реальных данных FSO-канала
