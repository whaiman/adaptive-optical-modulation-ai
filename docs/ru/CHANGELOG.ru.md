# История изменений

> 🌐 [English](../../CHANGELOG.md) · [Azərbaycanca](../az/CHANGELOG.az.md)

## [Не выпущено]

### Изменено

- Полная переработка архитектуры нейронной сети
- Задача переформулирована в отдельные этапы регрессии и классификации
- Фокус проекта уточнён в сторону оптики свободного пространства (FSO)
- Ведётся исправление концептуальных ошибок, выявленных в начальном прототипе
- Возможности модели расширены для соответствия современным стандартам исследований в области FSO и адаптивной модуляции

### Удалено

- Архитектура начального прототипа из-за концептуальных ограничений
